Troubleshooting Guide

Step-by-step solutions to common issues

üîß Quick Fixes (Try These First)

Before diving into specific issues, try these common solutions that resolve most problems:

  1. Force close and restart the Finvinity app
  2. Check your internet connection (prices require internet access)
  3. Pull down to refresh on the main portfolio screen
  4. Update the app to the latest version from your app store
  5. Restart your device if issues persist

üí° Pro Tip: These steps resolve about 80% of reported issues!

üì± App Performance Issues

App Won't Open or Crashes on Startup

Symptoms: App closes immediately, black screen, or stuck on loading screen

Solutions:

  1. Force Close App:
    • iOS: Double-tap home button (or swipe up on newer iPhones), swipe up on Finvinity
    • Android: Recent apps button, swipe away Finvinity
  2. Restart Device: Power off completely, wait 10 seconds, power back on
  3. Check Storage: Ensure at least 100MB free space on your device
  4. Update App: Check App Store/Play Store for available updates
  5. Reinstall App: Delete and reinstall (‚ö†Ô∏è This will delete all portfolio data - export first!)

App Runs Slowly or Freezes

Symptoms: Delayed responses, UI freezing, slow navigation

Solutions:

  1. Close Other Apps: Free up RAM by closing background apps
  2. Check Device Age: Older devices (4+ years) may experience slower performance
  3. Clear Cache:
    • iOS: Restart the app
    • Android: Settings ‚Üí Apps ‚Üí Finvinity ‚Üí Storage ‚Üí Clear Cache
  4. Reduce Portfolio Size: Large portfolios (50+ items) may slow performance
  5. Check Internet Speed: Slow connections can affect app responsiveness

High Battery Usage

Symptoms: Rapid battery drain when using Finvinity

Solutions:

  1. Update to Latest Version: Recent updates optimize battery usage
  2. Close App When Not in Use: Don't leave it running in background unnecessarily
  3. Check Background Refresh:
    • iOS: Settings ‚Üí General ‚Üí Background App Refresh ‚Üí Finvinity (turn off)
    • Android: Settings ‚Üí Apps ‚Üí Finvinity ‚Üí Battery ‚Üí Background Activity (restrict)
  4. Reduce Refresh Frequency: Manual refresh instead of frequent automatic updates

üìä Data and Price Issues

Prices Not Updating

Symptoms: Old prices showing, "Last updated" timestamp not recent

Solutions:

  1. Manual Refresh: Pull down on portfolio screen to force update
  2. Check Internet: Verify strong Wi-Fi or cellular connection
  3. Verify Market Hours:
    • Stock markets: Closed on weekends and holidays
    • Crypto markets: 24/7 but may have brief maintenance windows
    • Forex markets: Closed Friday evening to Sunday evening
  4. Check Asset Status: Some assets may be delisted or suspended
  5. Wait and Retry: API servers may be temporarily unavailable

Wrong or Inaccurate Prices

Symptoms: Prices that seem incorrect compared to other sources

Solutions:

  1. Verify Asset Symbol: Ensure you selected the correct ticker (e.g., AAPL vs AAPLF)
  2. Check Exchange: Same company may trade on different exchanges with different prices
  3. Consider Delays: Some data feeds have 15-20 minute delays
  4. Currency Differences: Check if price is displayed in different currency
  5. Compare Sources: Cross-reference with Yahoo Finance, Google Finance
  6. Report Issue: If persistently wrong, contact support with details

Can't Find Specific Assets

Symptoms: Search returns no results for your stock/crypto

Solutions:

  1. Try Different Search Terms:
    • Ticker symbol (AAPL)
    • Full company name (Apple Inc)
    • Alternative symbols (some companies have multiple listings)
  2. Check Spelling: Verify correct spelling of ticker/company name
  3. Verify Public Trading: Ensure the asset is publicly traded
  4. Check Asset Type: Make sure you're searching in the right category
  5. Try Partial Search: Search for just part of the name
  6. Request Addition: Contact us to add missing assets

üíº Portfolio Management Issues

Portfolio Data Missing or Lost

Symptoms: Empty portfolio, investments disappeared

‚ö†Ô∏è Prevention is Key: Regular data exports are essential since data is stored locally only

Recovery Options:

  1. Check for Backup: Look for exported CSV/JSON files
  2. Restart App: Sometimes data appears to be missing due to loading issues
  3. Avoid Multiple Devices: Data doesn't sync between devices
  4. Import Previous Export: If you have backup files, use Settings ‚Üí Import Data
  5. Re-enter Manually: If no backup exists, data must be re-entered

Incorrect Calculations

Symptoms: Wrong gains/losses, percentage calculations seem off

Solutions:

  1. Verify Purchase Data: Check quantity, price, and date are correct
  2. Check Currency Settings: Ensure base currency is set correctly
  3. Review Multiple Entries: Same stock bought multiple times affects averages
  4. Consider Fees: App doesn't include brokerage fees in calculations
  5. Refresh Current Prices: Outdated prices affect calculations
  6. Manual Verification: Double-check math with calculator if needed

Can't Edit or Delete Investments

Symptoms: Edit button not working, delete option missing

Solutions:

  1. Tap Investment Row: Tap directly on the investment to edit
  2. Swipe Actions: Try swiping left on investment for delete option
  3. Check App Update: Ensure you have latest version with all features
  4. Restart App: UI elements may be unresponsive
  5. Long Press: Try long-pressing investment for additional options

üåê Network and Connectivity Issues

No Internet Connection Error

Symptoms: "No connection" messages, data won't load

Solutions:

  1. Check Network Connection:
    • Wi-Fi: Verify connected to working network
    • Cellular: Check signal strength and data allowance
  2. Test Other Apps: Verify internet works in browser or other apps
  3. Reset Network Settings:
    • iOS: Settings ‚Üí General ‚Üí Reset ‚Üí Reset Network Settings
    • Android: Settings ‚Üí System ‚Üí Reset ‚Üí Reset Network Settings
  4. Switch Networks: Try different Wi-Fi network or cellular
  5. Firewall/VPN: Check if firewall or VPN is blocking connections

Slow Data Loading

Symptoms: Long loading times, timeouts

Solutions:

  1. Check Internet Speed: Use speed test app to verify connection quality
  2. Switch to Wi-Fi: Use Wi-Fi instead of cellular for faster speeds
  3. Close Background Apps: Free up bandwidth by closing other apps
  4. Try Later: API servers may be experiencing high traffic
  5. Reduce Portfolio Size: Fewer assets = faster loading

üì± Device-Specific Issues

iOS-Specific Issues

App Store Download Problems

  1. Check iOS Version: Requires iOS 13.0 or later
  2. Storage Space: Need at least 50MB free space
  3. Apple ID Issues: Sign out and back into App Store
  4. Region Restrictions: App may not be available in all countries

iOS Permission Issues

  1. Notifications: Settings ‚Üí Notifications ‚Üí Finvinity
  2. Cellular Data: Settings ‚Üí Cellular ‚Üí Finvinity (enable)
  3. Background Refresh: Settings ‚Üí General ‚Üí Background App Refresh

Android-Specific Issues

Google Play Download Problems

  1. Check Android Version: Requires Android 7.0 or higher
  2. Clear Play Store Cache: Settings ‚Üí Apps ‚Üí Google Play Store ‚Üí Storage ‚Üí Clear Cache
  3. Google Account: Ensure signed into correct Google account
  4. Device Compatibility: Some older devices may not be compatible

Android Permission Issues

  1. Network Access: Settings ‚Üí Apps ‚Üí Finvinity ‚Üí Permissions
  2. Storage Access: Required for data export/import features
  3. Background Activity: Settings ‚Üí Apps ‚Üí Finvinity ‚Üí Battery ‚Üí Background Activity

Avoid common issues with these best practices:

Data Protection

  • Regular Backups: Export your data weekly using Settings ‚Üí Export Data
  • Cloud Storage: Save backup files to Google Drive, iCloud, or Dropbox
  • Multiple Formats: Export in both CSV and JSON formats
  • Test Imports: Periodically test that your backups work correctly

App Maintenance

  • Keep Updated: Install app updates promptly for bug fixes and improvements
  • Restart Regularly: Restart the app occasionally to clear memory
  • Monitor Storage: Keep at least 100MB free space on your device
  • Stable Internet: Use reliable internet connections for data updates

Portfolio Management

  • Double-Check Entries: Verify ticker symbols and purchase details
  • Reasonable Size: Keep portfolios under 50 items for optimal performance
  • Regular Reviews: Periodically review and clean up old or incorrect entries
  • Note-Taking: Use the notes field to document important details
